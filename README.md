# Canteen Management System

---

## Overview
The **Canteen Management System** is a web-based mini project developed using **HTML, CSS, JavaScript, PHP, and MySQL**. It is designed to simplify and automate canteen operations such as managing menu items, placing orders, and handling basic administrative tasks.

This project was developed as part of a **Web Technology and DBMS course mini project**.

---

## Features
- User-friendly interface for customers  
- Display of available food items  
- Add items to cart and place orders  
- Admin panel to manage menu items  
- Add, update, and delete food items  
- Database integration using MySQL  
- Basic form validation using JavaScript  

---

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL
